Le dossiert "Simulateur Institut" est le projet unity.
Dans les assets il y a différent dossier qui sont :
- car -> Le modèle de la voiture.
- InterfaceManager -> Mon package de gestion Input/Output
- OVR -> Le module de l'occulus.
- Plugins -> plugins pour l'occulus
- RoadKit -> La route
- Scene -> La scène
- Script -> le script du comportement de la voiture.
- Standard Assets -> les arbres.

C'est normal que la voiture ne bouge pas si vous lancez la scène. Les inputs sont
actuellement configurés pour le simulateur Logitech.
